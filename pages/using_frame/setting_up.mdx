# Setting up a wallet

### Installing a wallet

You can use Frame with any Ethereum wallet. Here are some popular options to get you started:

- [MetaMask](https://metamask.io/)
- [Coinbase Wallet](https://wallet.coinbase.com/)
- [Trust Wallet](https://trustwallet.com/)
- [Rabby Wallet](https://rabby.io/)
- [Taho](https://taho.xyz/)

For a more comprehensive breakdown, you can read more about wallets [here](https://ethereum.org/en/wallets/find-wallet/).

### Switching Networks

To switch to Frame's network, simply click on the button below once you have a wallet installed.
